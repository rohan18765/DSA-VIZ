C++ Implementation 
int partition(int arr[], int low, int high) {
    int pivot = arr[high]; 
    int i = (low - 1); 
  
    for (int j = low; j <= high - 1; j++) {
        if (arr[j] < pivot) {
            i++; 
            swap(arr[i], arr[j]);
        }
    }
    swap(arr[i + 1], arr[high]);
    return (i + 1);
}

void quickSort(int arr[], int low, int high) {
    if (low < high) {
        int pi = partition(arr, low, high);
        quickSort(arr, low, pi - 1);
        quickSort(arr, pi + 1, high);
    }
}

Time Complexity

O(n log n)

Average case. Worst case O(n²) if pivot is bad.

Space Complexity

O(log n)

Stack space for recursion.

Pro Tip: Quick Sort is often faster in practice than Merge Sort because it has better cache locality and doesn't need extra memory for arrays, but it isn't stable.
