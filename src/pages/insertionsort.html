C++ Implementation 
void insertionSort(int arr[], int n) {
    for (int i = 1; i < n; i++) {
        int key = arr[i];
        int j = i - 1;

        // Move elements greater than key
        // to one position ahead
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j = j - 1;
        }
        arr[j + 1] = key;
    }
}

Time Complexity

O(n²)

Worst & Average case. O(n) if already sorted.

Space Complexity

O(1)

In-place algorithm.

Pro Tip: Insertion sort is efficient for small data sets or arrays that are already substantially sorted. It's often used as the base case for complex algorithms like QuickSort.
