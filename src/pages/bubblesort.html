C++ Logic 
            #include <iostream>
            using namespace std;

            void bubbleSort(int arr[], int n) {
                for (int i = 0; i < n - 1; i++) {
                    bool swapped = false;

                    for (int j = 0; j < n - i - 1; j++) {
                        if (arr[j] > arr[j + 1]) {
                            swap(arr[j], arr[j + 1]);
                            swapped = true;
                        }
                    }

                    if (!swapped)
                        break;
                }
            }

Time Complexity

  • Worst Case O(n²)
  • Average Case O(n²)
  • Best Case O(n)

Space Complexity

O(1) (Auxiliary)
